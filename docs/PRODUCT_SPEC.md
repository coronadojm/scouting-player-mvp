# Especificación MVP

## Objetivo

Aplicación para seguimiento individual de un jugador en vídeo de fútbol base, con análisis técnico, táctico, físico y formativo.

## Usuario principal

- Entrenador de fútbol base
- Scout
- Coordinador de cantera
- Padre/madre con supervisión técnica

## Datos del jugador

- Nombre
- Edad
- Categoría
- Posición
- Pierna dominante
- Nivel

## Métricas MVP

### Técnica
- Control orientado
- Pase
- Conducción
- Regate
- Pérdidas

### Táctica
- Posicionamiento
- Apoyos
- Toma de decisión
- Reacción tras pérdida

### Física
- Intensidad
- Aceleraciones estimadas
- Cambios de ritmo
- Participación

### Formativa
- Fortalezas
- Aspectos a mejorar
- Ejercicios recomendados

## Limitaciones

El MVP genera un informe simulado basado en perfil. Para producción, el backend debe integrar modelos de visión por computador y validación humana.
